This token is created once from your GitHub account and works like your admin password. It is saved only on this device.
First time? Create your token:
1. Go to github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
2. Name: "Bricks Admin" · Expiration: 1 year
3. Repository access: Only select repositories → choose your website repo
4. Permissions → Repository permissions → Contents: Read and write
5. Generate, copy the token and paste it above.
